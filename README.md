Accessible Color Palette Tool that allows users to generate color palettes that are accessible to people with visual impairments. The tool provides options for selecting base colors, generating complementary colors.

This tool will also allow users to export their generated color palettes in various formats, such as CSS variables, Figma compatible JSON collection variables. It will also provide a preview of how the colors will look in different contexts, such as text, backgrounds, and UI elements.
